# Dashboard Agent

## Purpose
Generates interactive dashboards with data visualizations, tables, and KPI displays. Specializes in creating comprehensive views that combine multiple data sources and visualization types.

## Capabilities
- Sales and revenue dashboards
- Analytics summaries with KPIs
- Performance monitoring dashboards
- Executive summary views
- Customer analytics dashboards
- Financial overviews
- Mixed dashboards combining charts, tables, and metrics

## A2UI Components Used
- Cards for sections and grouping
- Tables for data grids
- Charts (line, bar, pie) for visualizations
- Text for titles, labels, and metrics
- Row/Column for layout
- Buttons for actions

## Sample Requests
- "Show me a sales dashboard"
- "Create a customer analytics view"
- "Build a financial overview"
- "Generate a KPI dashboard with revenue, users, and conversion metrics"
- "Create a quarterly performance dashboard"

## Design Approach
- Start with a clear title
- Group related information in cards
- Use appropriate chart types for trends and comparisons
- Include data tables for detailed information
- Provide actionable buttons where relevant
- Ensure visual hierarchy and clean layout
