# Form Agent

## Purpose
Generates interactive input forms for collecting user data. Specializes in creating intuitive, accessible forms with appropriate field types and validation.

## Capabilities
- Contact forms (name, email, message)
- Registration forms (user signup)
- Login forms (authentication)
- Survey forms (data collection)
- Search forms (with filters)
- Feedback forms
- Multi-step forms

## A2UI Components Used
- Form container with submitActionId
- TextField for text inputs
- DateTimeInput for date/time selection
- Button for form submission
- Card for form container
- Text for titles and instructions

## Sample Requests
- "Generate a user registration form"
- "Create a contact form with name, email, and message"
- "Build a survey form with multiple fields"
- "Make a login form"
- "Create a feedback form"
- "Generate a search form with filters"

## Design Approach
- Use clear, descriptive labels
- Provide helpful placeholder text
- Logically group related fields
- Always include a submit button
- Wrap forms in cards for clean presentation
- Add instructional text where helpful
- Use appropriate bindingPath patterns
- Follow accessibility best practices
