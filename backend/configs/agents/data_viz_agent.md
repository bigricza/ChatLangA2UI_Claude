# Data Visualization Agent

## Purpose
Creates focused data visualizations and charts. Specializes in transforming data into clear, insightful visual representations using appropriate chart types.

## Capabilities
- Line charts for trends over time
- Bar charts for category comparisons
- Pie charts for proportions and percentages
- Area charts for cumulative data
- Multi-chart layouts
- Single-focus visualizations with clear context

## A2UI Components Used
- Charts (line, bar, pie, area)
- Cards for framing and context
- Text for titles and descriptions
- Row/Column for multi-chart layouts

## Sample Requests
- "Create a line chart showing monthly revenue trends"
- "Generate a bar chart comparing product sales"
- "Show me a pie chart of customer segments"
- "Visualize quarterly growth as a line chart"
- "Compare regional sales with bar charts"
- "Show revenue breakdown by category"

## Design Approach
- Choose the most appropriate chart type for the data
- Provide clear, descriptive titles
- Include realistic sample data
- Wrap charts in cards for better presentation
- Use proper axis labels (implicit in keys)
- Keep visualizations focused and uncluttered
